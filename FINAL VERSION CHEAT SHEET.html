HTML/CSS Cheat Sheet

Basic HTML Elements
Element Name Description
Document Metadata
<html> HTML Document The top-level element of an HTML document. All other elements are encompasssed in this element
<head> Head Contains machine-readable info (title, scripts, styles sheets)
<link> Link Specifies relationships between the current document and an external source. Common uses include linking to CSS and linking other styles
<title> Title Defines document's title in browser's title bar on the tab at top of page
Sectioning Root / Content Sectioning
<body> Body Represents the content of an HTML document
<address> Address Enclosed HTML provides contact info for person, people, or organization
<article> Article Self-contained composition in a document, page, application, or site
<aside> Aside Represents a portion of a document whose content is only indirectly related to the document's main content
<footer> Footer Represents a footer for its nearest ancestor sectioning content/root element
<header> Header Represents intro content, typically a group of intro or navigational ads (logo, search form, author name, etc.)
<h1-h6> Section Headings Six levels of section headings: h1 is highest, h6 is lowest
<main> Main Represents the dominant content of the body of a document
<section> Section Represents a generic standalone section of a document
Text Content
<div> Division Generic container for flow content. Has no effect on content or layout until styled in CSS
<figcaption> Figure Caption Represents a caption or legend describing the rest of its parent content (see below)
<figure> Figure Represents self-contained content (parent element)
<p> Paragraph Represents a paragraph
HTML Basic Table Elements To Know
Element Name Description
<table> Table The wrapper element for all HTML tables
<thead> Table Head The set of rows defining he column headers in a table
<tbody> Table Body The set of rows containing actual table data
<tr> Table Row The table row container
<td> Table Data The table row container
<tfoot> Table Foot The set of rows defining the footer in a table
Table Attributes
Attribute Name Description
colspan Column Span Defines how many columns a td element should span
rowspan Row Span Defines how many rows a td element should span
